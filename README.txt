    Hola, este es un archivo para explicar el
correcto uso del programa ppm.
    El usuario ingresara en una consola
el comando "./ppm.sh <NAME>" donde
<NAME> es el nombre del proyecto a 
crear.
    Luego de esto se ejecutara una
serie de pasos automaticamente y
una serie de carpetas y archivos se
creara automaticamente con el
contenido deseado.
